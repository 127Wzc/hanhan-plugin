## 安装

### Github
> git clone https://github.com/hanhan258/hanhan-plugin.git ./plugins/hanhan-plugin

### Gitee
> git clone https://gitee.com/han-hanz/hanhan-plugin.git ./plugins/hanhan-plugin

## 安装依赖，到yunzai根目录下执行
```bash:numbers
cd ./plugins/hanhan-plugin
pnpm i
```

安装完依赖之后，重启机器人，就可以食用了
